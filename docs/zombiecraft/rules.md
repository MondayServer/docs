# 服务器规则

## 总则
1. 尊重其他玩家，禁止任何形式的辱骂、歧视、人身攻击
2. 禁止使用外挂、作弊客户端、BUG获利等不公平手段
3. 禁止恶意破坏服务器运行，如刷屏、卡服
4. 禁止现实货币交易游戏物品、账号

## ⚔️ 游戏行为
1. 禁止恶意PVP (后续考虑开设竞技场)
2. 允许合理争夺资源，但禁止守尸、堵门等恶意行为  
3. 禁止利用BUG复制物品、刷经验等
4. 禁止恶意破坏他人建筑
5. 禁止使用影响游戏平衡的模组（透视等）

## 🏠 建筑领地
1. 在他人领地建筑需获得所有者同意
2. 禁止建造不雅、违规内容建筑
3. 禁止圈城市建筑、烈焰人刷怪笼等公用结构

## 💎 经济交易
1. 交易请遵循自愿原则，禁止欺诈行为
2. 发现经济BUG请及时上报，隐瞒利用将严惩

## 👥 社交互动
1. 禁止不良言语、刷屏、广告
2. 组队合作请遵守承诺，禁止背信弃义
3. 新人玩家应获得帮助，禁止恶意欺压

## ⚠️ 违规处罚
初犯: 警告 + 物品回收
再犯: 禁言 1-7天  
严重: 封禁 7-30天
恶意: 永久封禁 + IP封禁

## 💡 补充说明
1. 管理员拥有最终解释权
2. 规则会根据服务器发展进行调整,请保持关注
3. 发现违规行为请及时向管理员举报
4. 合理利用游戏机制是允许的

**我们的目标**：创建一个公平、有趣、持久的僵尸末日世界！
